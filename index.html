Introduction.

This is a documentation page for the Python programming language. It is a good place to start if you're new to Python or just want to learn more about the language.

This documentation is written by a community volunteer and is in no way endorsed by the Python Foundation.

Why learn python?

Python is an interpreted, high-level, general-purpose programming language. Its design philosophy emphasizes code readability, and its syntax allows programmers to express concepts in fewer lines of code than would be possible in languages such as C++ or Java.
Python supports multiple programming paradigms, including object-oriented, imperative and functional.

I recommend following along with this tutorial so that you can get a hands-on experience with Python.

Basics

Basic functions

The -> is to show what the function returns returns

print(*args) -> None This function prints the given arguments to the console.

type(*args) -> type This function returns the type of the given argument.

input(str(prompt)) -> str This function returns the input from the user.

len(str() or list()) -> int This function returns the length of the given argument.

range(start, stop, step) -> range This function returns a list of numbers from the given start to the given end.

Variables

Variables are used to store values. Variables are declared using the name = value syntax.

Operators

Operators are used to perform operations on variables. Operators are declared using the operator syntax.

  • Math operations
    • Addition: +
    • Subtraction: -
    • Multiplication: *
    • Division: /
    • Modulus: %
    • Exponent: **
    • Floor Division: //
  • Binary
    • Bitwise AND: &
    • Bitwise OR: |
    • Bitwise XOR: ^
    • Bitwise NOT: ~
  • Comparison/Boolean operations
    • Logical AND: and
    • Logical OR: or
    • Logical NOT: not
    • Comparison: ==
    • Less than: <
    • Less than or equal to:<=
    • Greater than: >
    • Greater than or equal to: >=
  • Variable handling
    • Assignment: =
    • Addition assignment: +=
    • Subtraction assignment: -=
    • Multiplication assignment: *=
    • Division assignment: /=
    • Modulus assignment: %=
    • Exponent assignment: **=
    • Bitwise AND assignment: &=
    • Bitwise OR assignment: |=
    • Bitwise XOR assignment: ^=

Loops

Loops are used to repeat a block of code. Loops are declared using the for syntax or the while syntax.

For Loop

The for var in condition: loop is used to iterate over a sequence.

While Loop

The while condition: loop is used to iterate over a sequence until a condition is met.

Conditional Statements

Conditional statements are used to execute different code based on a condition. Conditional statements are declared using the if syntax.

If Statement

The if condition: statement is used to execute different code based on a condition. If the condition is true, the code in the if statement is executed.

Else Statement

The else: statement is used to execute different code if the condition is not met.

Elif Statement

The elif condition2: statement is used to execute different code based on a different condition if the condition in the if statement is not met.

Nested If Statements

The if statement can be nested inside of another if statement.

Functions

Functions are used to group code together. Functions are declared using the def funcName(arguments): syntax.

Return Statement

The return value statement is used to return a value from a function.

Arguments

Arguments are used to pass values into a function. Arguments are declared using the funcName(argument) syntax.

Keyword Arguments

Keyword arguments are used to pass values into a function. Keyword arguments are declared using the funcName(**keyword) syntax.

Default Arguments

Default arguments are used to pass default values into a function. Default arguments are declared using the funcName(argument = value) syntax.

Variable Length Arguments

Variable length arguments are used to pass values into a function. Variable length arguments are declared using the *args syntax.

Keyword Variable Length Arguments

Keyword variable length arguments are used to pass values into a function. Keyword variable length arguments are declared using the **kwargs syntax.

Classes

Classes are used to group variables together. Classes are declared using the class ClassName(): syntax.

__init__

The __init__ function is used to initialize a class, giving it all its variables.

Inheritance

Inheritance is used to create a new class based on an existing class. Inheritance is declared using the class ClassName(ParentClass): syntax.

Instance Variables

Instance variables are used to store values. Instance variables are declared using the self.variableName = value syntax.

Instances

Instances are used to create objects. Instances are declared using the objectName = ClassName() syntax.

Self

The self keyword is used to refer to the current instance of a class.

Static Variables

Static variables are used to store values. Static variables are declared using the ClassName.variableName = value syntax.

Static Methods

Static methods are used to group code together. Static methods are declared using the
@staticmethod
def methodName():
 syntax. Static methods are called by running ClassName.methodName()

Instance Methods

Instance methods are used to group code together. Instance methods are declared using the def methodName(self): syntax. Instance methods can be called by using the objectName.methodName() syntax.

Exceptions

Exceptions are used to handle errors. Exceptions are declared using the try: syntax. The code in the try: will run

Raising Exceptions

The raise keyword is used to raise an exception.

Exception Handling

The except: keyword is used to handle an exception that is raised from the code in try:.

else

The else: keyword is used to execute different code if an exception is not raised.

Finally

The finally: keyword is used to execute code after an exception has been handled.

Iterators

Iterators are used to iterate through a list of values. Iterators are declared using the for value in list: syntax.

Data types

Integers

Integers are whole numbers. They are the most basic type of data in Python.

An example of an integer is, simply, '1'. An integer is any whole number without any decimal points. This excludes 1.0, which is a floating point number. When defining a variable with a value of an integer, you do not need any quotations or brackets. It will make it another data type.


The statement below will give an output of '1'

print(1)

Output:

1


The next statement here will print the data type of one.

print(type(1))

Output:

<class 'int' >

As we can see here, the number 1 is an integer.

The same output will be printed if we give another whole number. However, if we try a number with a decimal point or with quotes, it will not be an integer.

print(type(1.0))

Output:

<class 'float' >

As we can see here, the number 1.0 is a floating point number.
To turn a float into an int, we have to call the int(floatNumber) function. This will remove all the leading digits after the decimal point and give only the whole number

If we write the number in quotes, it will be a string.

print(type("1"))

Output:

<class 'str' >

As we can see here, the number 1 is a string as it is wrapped in quotes.

An example of a float is, simply, '1.0'. A float is any number with a decimal point. This includes 1.0, which is an integer. When defining a variable with a value of a float, you do not need any quotations or brackets. It will make it another data type.

Output:

As we can see here, the number 1.0 is a float.

The same output will be printed if we give another whole number. However, if we try a number with a decimal point or with quotes, it will not be an integer.

Output:

<class 'int' >

As we can see here, the number 1 is an integer. We can turn this number into a float by using the float() function

The following will give an output of 1.0

Tuple

Tuples are similar to lists, but they are immutable. Tuples are used when you want to store a list of values that you do not want to change.

Tuples are created by using the parenthesis () and separating the values with a comma.

The following will create a tuple with 3 values
a = (1, 2, 3)
print(a)

Output:

(1, 2, 3)

The following will add an item to a tuple
a = (1, 2, 3)
a = a + (4,)
print(a)

Output:

(1, 2, 3, 4)

the following will add an item to a tupple using append()
a = (1, 2, 3)
a.append(4)
print(a)

Output:

AttributeError: 'tuple' object has no attribute 'append'

The following will remove an item from a tuple
a = (1, 2, 3)
a.remove(2)
print(a)

Output:

(1, 3)

Why use tuples over lists

Tuples are faster than lists. Lists are mutable, while tuples are immutable.

Tuples are used when you want to store a list of values that you do not want to change.

Set

Sets are similar to lists, but they are unordered. Sets are used when you want to store a list of values that you do not want to change.

Sets are created by using the curly brackets {} and separating the values with a comma.

The following will create a set with 3 values
a = {1, 2, 3}
print(a)

Output:

{1, 2, 3}

The following will add an item to a set
a = {1, 2, 3}
a.add(4)
print(a)

Output:

{1, 2, 3, 4}

The following will remove an item from a set
a = {1, 2, 3}
a.remove(2)
print(a)

Output:

{1, 3}

The following will remove an item from a set using discard()
a = {1, 2, 3}
a.discard(2)
print(a)

Output:

{1, 3}

Why use sets over other array types

  • Sets automatically order the values in it
  • Sets remove duplicates

a = {}
print(type(a))
The data type of variable a is a dictionary

Output:

<class 'dict>

To create an empty set, you will have to use the set() function.

a = set()
print(type(a))
The data type of a is a set

Output:

<class 'set>

Dictionary

Dictionaries are similar to lists, but they are unordered. Dictionaries are used when you want to give custom indexes to a value. Dictionaries are made up for keys and values.

Dictionaries are created by using the curly brackets {key: value} and separating the key:values with a comma.

The following will create a dictionary with 3 values
a = {1: 1, 2: 2, 3: 3}
print(a)

Output:

{1: 1, 2: 2, 3: 3}

The following will add an item to a dictionary
a = {1: 1, 2: 2, 3: 3}
a[4] = 4
print(a)

Output:

{1: 1, 2: 2, 3: 3, 4: 4}

The following will remove an item from a dictionary
a = {1: 1, 2: 2, 3: 3}
del a[2]
print(a)

Output:

{1: 1, 3: 3}

The following will remove an item from a dictionary using pop()
a = {1: 1, 2: 2, 3: 3}
a.pop(2)
print(a)

Output:

{1: 1, 3: 3}

The following will remove an item from a dictionary using popitem()
a = {1: 1, 2: 2, 3: 3}
a.popitem()
print(a)

Output:

{2: 2, 3: 3}

The following will attempt to remove a key in a dictionary that isn't in the dictionary
a = {1: 1, 2: 2, 3: 3}
a.pop(4)
print(a)

Output:

KeyError: 4

The following will get the key given the value of the key
a = {1: 1, 2: 2, 3: 3}
print(a[2])

Output:

2
